###### Likelion Front-end Bootcamp

# HTML/CSS

멋쟁이 사자처럼 프론트엔드 부트캠프 HTML/CSS 학습 저장소

## 학습 저장소 복제하기

```sh
git clone https://github.com/seulbinim/learn-html-css.git
```

## 복제한 저장소로 이동하기

```sh
cd
cd bootcamp/learn-html-css
```

## 복제한 저장소에 등록된 Remote 정보 삭제하기

```sh
git remote rm origin
```

## 의존성 패키지 설치하기

```sh
npm install
```

## 학습 저장소를 Github에 백업하기

```sh
1. VS Code의 소스 제어(Source Control) 항목 선택
2. Publish Branch 버튼 클릭
3. Publish to Github public repository 클릭
```

## 학습 저장소에 새로운 커밋 생성 시 Github에 반영하기

```sh
git push
```
